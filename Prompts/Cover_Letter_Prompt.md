# Cover Letter Generation Rules

Generate highly customized professional cover letter.

Requirements:
- concise
- recruiter-friendly
- no generic filler
- no fake claims
- strong ATS alignment
- mention matching technologies
- mention automation experience
- mention framework development
- mention API testing
- mention AI learning direction
- professional tone

Avoid:
- overly emotional language
- generic AI phrases
- excessive length
- fake achievements

Focus On:
- Selenium
- API testing
- Appium
- framework development
- CI/CD
- automation architecture
